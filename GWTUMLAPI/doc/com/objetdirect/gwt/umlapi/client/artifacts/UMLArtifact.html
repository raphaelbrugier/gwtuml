com.objetdirect.gwt.umlapi.client.artifacts
Class UMLArtifact

Object
  UMLArtifact
Direct Known Subclasses:
BoxArtifact, LinkArtifact
public abstract class UMLArtifact
extends Object

This abstract class represent any uml artifact that can be displayed An artifact is something between the graphical object and the uml component.
It has an uml component and it build the graphical object

Author:
Florian Mounier (mounier-dot-florian.at.gmail'dot'com)
